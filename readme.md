# xdotter: manage dotfiles like .zshrc, .vimrc, .gitconfig,.bashrc ...

## Install

Use cargo-binstall:

```zsh
cargo binstall xdotter
```

Use cargo:

```zsh
cargo install xdotter
```

From source:

```zsh
git clone github.com/cncsmonster/xdotter
cd xdotter
cargo install --path .
echo 'xdotter is already installed in .cargo/bin/xdotter.'
```
